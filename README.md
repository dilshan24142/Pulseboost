# PulseBoost - Windows System Optimizer

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![.NET Version](https://img.shields.io/badge/.NET-6.0-blue)](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)

**PulseBoost** is a powerful yet lightweight Windows Forms application built with C# and .NET 6.0. It is designed to help users optimize their PC's performance, monitor system resources in real-time, and enhance the gaming experience through an intuitive graphical interface.

## ✨ Features

*   **Real-Time System Monitoring**: Keep an eye on your critical system resources with live performance graphs and data for:
    *   **CPU** Usage
    *   **RAM** Consumption
    *   **Network** Activity
*   **One-Click Gaming Optimization**: Instantly boost your gaming performance by automatically closing unnecessary background applications and services to free up valuable CPU and RAM.
*   **Customizable Optimization Profiles**: Create, save, and load your own sets of optimization rules. Tailor the software's behavior to match your specific workflow or gaming needs.
*   **Lightweight & Fast**: Built for efficiency, PulseBoost has a minimal impact on your system's resources while it's running, ensuring your PC stays fast.
*   **User-Friendly Interface**: The application is built with Windows Forms, providing a clean, simple, and easy-to-navigate experience for users of all levels.
