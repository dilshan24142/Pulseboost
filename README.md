# PulseBoost - Exoptimizer  Windows Form Application

**PulseBoost** is a Windows Forms application built with C# and .NET 6.0.  
It is designed to **optimize PC performance**, monitor system resources, and enhance gaming experience with an easy-to-use interface.

---

## Features

- **CPU, RAM, and Network Monitoring** – Real-time performance metrics
- **Gaming Optimization** – Close unnecessary applications to free system resources
- **Custom Profiles** – Create custom optimization settings
- **Lightweight and Fast** – Minimal impact on system performance
- **User-Friendly GUI** – Built using Windows Forms for easy navigation
